# 급여 계산기

급여 정산 금액을 자동으로 계산해주는 웹 애플리케이션입니다.

## 기능

- 총매출 입력
- 수수료 자동 계산 (85%)
- 인센티브율 설정 및 계산
- 재료비 자동 계산 (10%)
- 인턴비 입력
- 세금 자동 계산 (3.3%)
- 최종 정산 금액 표시

## 계산 순서

1. 총매출에서 수수료 85% 차감
2. 남은 금액에서 인센티브 차감
3. 재료비 10% 차감
4. 인턴비 차감
5. 세금 3.3% 차감
6. 최종 정산 금액 산출

## 기술 스택

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Zustand (상태 관리)

## 시작하기

개발 서버 실행:

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 열기

## 프로젝트 구조

```
calculator/
```

## 상태 관리

Zustand를 사용하여 계산기의 모든 상태를 중앙에서 관리합니다:
- 입력값 (총매출, 인센티브율, 인턴비)
- 계산 결과 (수수료, 재료비, 세금, 정산금액)
